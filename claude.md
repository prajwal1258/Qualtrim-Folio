# Seaside Retreats Project Rules

This project follows the "Essence of Nature" design system and strict minimalist architecture.

## Project Structure
- `index.html`: Main landing page
- `style.css`: Core design system and component styles
- `claude.md`: Architectural overview
- `.claude/rules/`: Detailed component and design rules

## Guidelines
- Use semantic HTML5 exclusively.
- Follow the design tokens defined in the design system.
- Maintain a mobile-first responsive approach.
